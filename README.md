# あついよ〜！わんこレスキュー

特別支援学校の生活単元学習「天気・暑さ・涼しくする工夫」の導入で使うiPad向けWebゲームです。

## 使い方

1. `index.html`、`styles.css`、`app.js`、`assets` フォルダをGitHubリポジトリ直下に置く
2. GitHub Pagesを有効にする
3. 公開URLをiPadで開く

## 重要

リポジトリ直下が次の形になるようにしてください。

```text
rescue-seitan/
├── index.html
├── styles.css
├── app.js
├── assets/
│   ├── backgrounds/
│   ├── dogs/
│   ├── items/
│   └── ui/
└── README.md
```

`rescue-seitan/rescue-seitan-ready/index.html` のように、フォルダが一段深くなると、想定したURLで開けないことがあります。

## 機能

- プレイヤー名入力
- 温度・げんきメーター表示
- アイテムタブ
- アイテムごとの「どう使う？」選択
- 背景・犬の状態変化
- 今日のランキング保存（同じ端末・同じブラウザ内）

## 更新メモ

- カーテンを何度も閉める、窓を何度も開けるなど、現実には連続してできない操作を制限しました。
- カーテンは「閉めたら次は開ける」、窓は「開けたら次は閉める」のように状態に応じて選択肢が変わります。
- できない選択肢は灰色で表示され、押せないようになっています。


## 今回の更新

- 窓・カーテン・エアコン・扇風機など、同じ操作を何度も繰り返して温度や点数を稼げないようにしました。
- できない選択肢は灰色になり、押せません。
- アイテムカードをドラッグして、わんこの上や部屋の中ではなすと「どう使う？」が開きます。
- ドラッグした場所に、使ったアイテムが小さく表示されます。

## 2026-06-07 追加修正
- カーテン・窓・扇風機・エアコンなどは、状態を持つようにしています。
- 開け閉めやモード切替で点数を稼げないように、効果は「累積」ではなく「置き換え」で計算します。
  - 例：カーテンを閉める → 涼しくなる効果
  - その後カーテンを開ける → その効果を取り消す
  - 同じ行動を何度も繰り返すことはできません。
- `assets/actions/` に、ドラッグ後に表示する「わんこが使っている」結果イラストを追加しました。
- `assets/action_sheets/` に、元の素材シートも保存しています。

## 今回の調整
- 扇風機とエアコンは別の道具として扱います。
- 扇風機を置いてもエアコンは消えません。
- エアコンを置いても扇風機は消えません。
- 同じ種類の道具だけが置き換わります。
  - 例：扇風機の向きを変えると、前の扇風機表示だけが置き換わります。
  - 例：エアコンを冷房から暖房に変えると、前のエアコン表示だけが置き換わります。
