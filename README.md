# あついよ〜！わんこレスキュー

特別支援学校の生活単元学習「天気・暑さ・涼しくする工夫」の導入で使うiPad向けWebゲームです。

## 使い方

1. `index.html`、`styles.css`、`app.js`、`assets` フォルダをGitHubリポジトリ直下に置く
2. GitHub Pagesを有効にする
3. 公開URLをiPadで開く

## 重要

リポジトリ直下が次の形になるようにしてください。

```text
rescue-seitan/
├── index.html
├── styles.css
├── app.js
├── assets/
│   ├── backgrounds/
│   ├── dogs/
│   ├── items/
│   └── ui/
└── README.md
```

`rescue-seitan/rescue-seitan-ready/index.html` のように、フォルダが一段深くなると、想定したURLで開けないことがあります。

## 機能

- プレイヤー名入力
- 温度・げんきメーター表示
- アイテムタブ
- アイテムごとの「どう使う？」選択
- 背景・犬の状態変化
- 今日のランキング保存（同じ端末・同じブラウザ内）

## 更新メモ

- カーテンを何度も閉める、窓を何度も開けるなど、現実には連続してできない操作を制限しました。
- カーテンは「閉めたら次は開ける」、窓は「開けたら次は閉める」のように状態に応じて選択肢が変わります。
- できない選択肢は灰色で表示され、押せないようになっています。
