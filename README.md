# あついよ〜！わんこレスキュー

生活単元学習「天気・暑さ」の導入で使える、iPad向けのWebゲームです。  
暑くてしんどそうな犬にアイテムを使い、温度やげんきメーターの変化を見ながら、暑い時の工夫を考えます。

## 追加した機能

- 最初にプレイヤー名を入力できます。
- 画面右に「今日のスコア」が表示されます。
- 「記録する」を押すと、その日のランキングに保存されます。
- 「ランキング」から、今日の順位を確認できます。
- 日付ごとに記録が分かれます。

## スコアの考え方

スコアは「快適ポイント」です。

- わんこの元気が高いほど点数が上がります。
- 部屋の温度が下がるほど点数が上がります。
- 28℃以下、30℃以下になるとボーナスが入ります。

授業では、ランキングを見ながら、

> どの使い方がよかったかな？  
> 温度は下がったかな？  
> げんきメーターは上がったかな？

と振り返る使い方がおすすめです。

## 注意：ランキングについて

このランキングは、**同じiPad・同じブラウザの中だけ**に保存されます。  
同じiPadを順番に使う授業なら、その日の対決として使えます。

複数のiPadで同じランキングを共有したい場合は、Firebaseなどのデータベース連携が必要です。

## 使い方

1. GitHubにこのフォルダの中身をアップロードします。
2. GitHub Pagesを有効化します。
3. iPadのSafariで公開URLを開きます。
4. プレイヤー名を入力します。
5. アイテムを選び、「どう使うか」を選びます。
6. 最後に「記録する」を押して、ランキングを確認します。

## ファイル構成

- `index.html`：画面本体
- `styles.css`：見た目
- `app.js`：ゲーム処理
- `README.md`：説明

## ゲームの特徴

- アイテムタブを開くと、たくさんのアイテムが出ます。
- 扇風機もアイテムに入っています。
- アイテムを選んだ後、「どう使うか」を選びます。
- お茶、水、毛布、ドライヤー、エアコンなどは、使い方によって結果が変わります。
- 青・赤で正解不正解を事前に分けず、結果で考えられるようにしています。

## 授業でのつなぎ方

ゲーム後に、次のように実際の活動につなげられます。

> ゲームでは、道具の使い方によって、すずしくなったり、逆に暑くなったりしました。  
> では、2組教室や5組教室では、どんな道具をどう使うと温度を下げられるでしょうか。

## 安全面の注意

- 実際の熱中症対応としては、涼しい場所への移動、水分補給、大人へ知らせることを優先してください。
- 実際の活動では、温度計に保冷剤や氷を直接当てないようにしてください。
- 水を使う場合は、床が滑らないように注意してください。
